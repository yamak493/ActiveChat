# Minecraftプラグイン『ActiveChat』制作方針
このプラグインは大規模なマルチサーバーにおいて、参加者さんのチャットを活性化するために作成します。
PaperMCが開発しているFoliaに対応してください。

## ポイント獲得条件
ポイントの獲得条件を満たしてもプレイヤーにその都度通知する必要はありません。次のログイン時に合計の獲得ポイントを表示＆付与するだけです。
### 1. 通常チャット
- 1回チャットするごとに1ポイント
- 間隔が3秒未満または文字数が6文字以下の場合はカウントしない

### 2. 新規さん歓迎チャット
- 「082」「%初%」「%新規さん%」「%よろしく%」とチャットするごとに50ポイント
- 間隔が30秒未満の場合はカウントしない
- サーバーに新規さんが参加（`player.hasPlayedBefore()`関数を使用して判定）してから60秒以内のみカウントする
※「%」はワイルドガードを意味します。

### 3. 挨拶チャット
- 「08」「52」「58」「%おは%」「%こん%」とチャットするごとに5ポイント
- 間隔が30秒未満の場合はカウントしない
- サーバーに誰かしらプレイヤー（直近5分以内に退出したプレイヤーを除く）が参加してから60秒以内のみカウントする
※「%」はワイルドガードを意味します。

## ポイント獲得のタイミング
プレイヤーごとの獲得したポイントは、jsonに保管（※）されます。
次回にサーバーに参加したときに、ポイントが獲得できます。
言い換えれば、サーバーに参加したときに、前回獲得したポイントが貰えます。

プレイヤーにポイントを与えるコマンドは、
```
/points give プレイヤー名 ポイント数
```
です。

※サーバーが再起動してもデータが消えないように

## プラグインの初期化
そのサーバーで初めてプラグインを起動するときは、jsonが存在しないので自動作成してください。
